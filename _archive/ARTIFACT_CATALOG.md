# Artifact Catalog

## Purpose
Lists the artifact types used in the Qenki ecosystem and the schema specialization rules they require.

## Artifact Types
- ADR
- Registry
- Standard
- Policy
- Specification
- Runbook
- Architecture
- Canon

## Type Requirements
Each type declares:
- which schema it uses
- which fields are mandatory
- which additional constraints apply

## Current Machine Contract Guidance
Artifact types should explicitly declare whether they are structured, partially structured, or human readable only.
