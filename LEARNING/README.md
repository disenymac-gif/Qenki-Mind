# Learning Repository

## Purpose
Learning entities live here as first-class persistent knowledge objects, recording resolved Predictions (confirmed or disconfirmed) derived from Expressions. Each entity is maintained in its own file, with global views used only as indexes or summaries.

## Ownership
Owned by the Learning & Reflection Organ per QENKI_MIND_ORGANS_v1.md. Only that organ may write the authoritative entity file.

## Entity-First Rule
- One file per learning entity is the preferred source of truth.
- Global logs, when present, are secondary views and must not be treated as the source of truth.
- Each entity may include context, hypotheses, predictions, consequences, learning, world-state links, memory links, and cross-entity links.

## Minimal Entity Structure
- Identity
- Ownership
- Canonical Basis
- Context
- Hypotheses
- Predictions
- Consequences
- Learning
- Links
- Current State
- Change History
- Last Updated

## Lifecycle
This repository evolves as ConsequenceToLearning generates new entities from resolved Predictions in Expressions. It is operational, not canonical architecture.
