# Evidence

## Purpose
Evidence sets capture the explicit support used by cognitive operators to justify downstream entities such as Decisions. Evidence is distinct from Memory: memory is retained knowledge, while evidence is the explicit support assembled for a specific transformation.

## Entity Structure
- Evidence Set ID
- Session
- Source Opportunity
- Evidence Items
- Ranked Order
- Supporting World State
- Supporting Objectives
- Supporting Memory
- Generated Hypotheses
- Confidence Estimate
- Last Updated
